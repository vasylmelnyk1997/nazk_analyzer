# mock web api to public api NAZK

all existed data are in '..\data' folder

root URL to mock: https://public-api.nazk.gov.ua/

- to get document URI /v2/documents/{doc_id}, where doc_id: UUID, the field "id" at the 0 level of json file

- to get list of documents URI /v2/documents/list?user_declarant_id={user_declarant_id}, where user_declarant_id: number, the field at the 0 level of json file
